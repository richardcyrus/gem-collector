How to Play

  1. The goal is to match the number shown above as the Target Number.
  2. Each of the gems below have a point value assigned to them.
  3. As you click on each gem, the points will be added to your total score.
  4. To win the game, you will need to use the points assigned to each gem to match the Target Number.
  5. You lose if you go over the Target Number.
  6. After each win or loss, the Target Number and the values assigned to each gem will change.
