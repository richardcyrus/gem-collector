How to Play

  • When the game begins, you'll be shown a number.
  • Each of the crystals below have a point value assigned to them.
  • To win the game, you will need to use the points assigned to each crystal to match the target amount.
  • You lose if you go over the target amount.
  • On each start of the game, the target number and the values assigned to each crystal will change.
